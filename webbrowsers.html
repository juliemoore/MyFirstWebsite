Web Browsers

A web browser is a software application you use to access the information from the World Wide Web, private networks or file systems, such as Windows File Explorer. The primary purpose and function of a browser is to retrieve information, display the information and then access other information by navigating through networks or links. Web browsers initiates the retrieval process when the user enters an internet address (URL). Web browsers use a layout engine to render pages from markup language, such as HTML, to a readable format.

The first web browser was invented by Sir Tim Berners-Lee in 1990. The first graphical user interface was Erase. In 1993, Marc Andreesen released Mosaic, which led to the explosion of the use of the internet. Andreesen later started his own company, named Netscape. In 1994, he released Netscape Navigator, which became the world's most popular browser, with a 90% market share.

The "browser war" began with the release of Microsoft's Internet Explorer in 1995. Opera was released in 1996 and focused their attention to mobile phone web browsers and the Nintendo Wii. In 1998, Netscape launched the Mozilla Foundation for using open source software to develop web browsers. Firefox is the result of the project. Apple Safari was released in 2003, primarily for Mac users. Chrome was released in 2008 and has since dominated the market. (Web browsers, n.d.) There are many other browsers available throughout the world. For informational purposes, I will be covering the following browsers:
